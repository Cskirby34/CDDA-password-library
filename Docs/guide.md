Password Manager Guide


This is a guide on how to use our password manager.

Step 1 - Creating your password vault
When starting up the password manager, you first have to make a vault. If this is your first time, it will ask you to make a password vault.

Step 4 - saving the vault
This step is simple, after adding an entry, you will have to save the vault to prevent it from being gone forever. Click the save vault to save and then you can press the lock button to close it afterwards. To open the vault againagain, type in the password for the vault.
